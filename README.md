### Install Mac OS / Linux

```sh
bash <(curl -s https://raw.githubusercontent.com/MaxwelMazur/tmux-setup/main/install.sh)
```

### Reload config tmux, to check for syntax failures
```bash
tmux source-file ~/.tmux.conf
```
