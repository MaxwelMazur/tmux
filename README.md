# tmux-setup

# Install

###Mac OS / Linux

```sh
bash <(curl -s https://raw.githubusercontent.com/)
```

- Installing TPM 
```bash
git clone https://github.com/tmux-plugins/tpm ~/.tmux/plugins/tpm
```

- Run, to check for syntax failures
```bash
tmux source ~/.tmux.conf
```

- Menu de janelas
```
Ctrl+b w
```

- Modo vim copiar e colar
```
Ctrl+[ enter Ctrl+]
```
